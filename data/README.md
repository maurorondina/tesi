# Information

- [LIST_PAPERS.txt](./LIST_PAPERS.txt) contains an ordered list of all downloaded papers
- [papers](./papers) is a folder containing the papers in their textual and xml versions. Some papers have xml versions that can be analyzed (suffix: '_analyzable.xml'): of these, some have versions used for the training set (suffix: '_analyzable_train.xml') or test set (suffix: '_analyzable_test.xml'). (The version '_analyzable_tmp.xml' must be ignored.)
- [paper-parquets](./paper-parquets) is a folder containing the data set produced in [AI_6-dataSet](../prediction_pd/AI_6-dataSet.ipynb)
- [paper-textclean-parquets](./paper-textclean-parquets) is a folder containing the text-clean data set produced in [AI_6-dataSet](../prediction_pd/AI_6-dataSet.ipynb)
